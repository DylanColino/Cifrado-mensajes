# Cifrado-mensajes
Trabajo practico grupal de la universidad, de la materia algoritmos y estructura de datos 1. Es sobre programación funcional en Haskell, y el objetivo del TP es realizar metodos de cifrado tales como el cifrado Cesar y el Vigenere, respetando especificaciones formales y restricciones propias de la materia

**Objetivos principales**
- Aplicar programacion funcional: Uso de recursion y pattern matching.
- No se permiten tecnicas avanzadas (como funciones de orden superior), ni estructura complejas fuera del listado aprobado.
- Seguir una especificacion precisa para cada funcion, con precondiciones y poscondiciones.

Este TP es una oportunidad para practicar **pensamiento declarativo** y trabajar con **funciones puras y recursivas,** en el contexto de criptografía clásica. Además, promueve buenas prácticas como testing, claridad del código y uso de sistemas de control de versiones.
